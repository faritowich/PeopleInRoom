# PeopleInRoom 
## Описание
Приложение представляет собок список с функцией добавления в него людей. В нём я объединил некоторые из изученных мной фич в разработке под Android.
## Используемый стек
*   Room
*   RecyclerView
*   Navigation Component
*   MVVM (ViewModel, LiveData)
*   ViewBinding
*   Coroutines
